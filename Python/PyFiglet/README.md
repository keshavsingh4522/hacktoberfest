# PyFiglet #

## Description ##

A python script that takes in ASCII text and renders it in various ASCII art fonts using the Pyfiglet library.


## Author ##

[ManinderAngural](https://github.com/ManinderAngural)

## How to use ##

Run the program

```console
$python PyFiglet.py
```

Output:

```
$ PyFiglet.py
Enter some text : 
```

```
Maninder
```
